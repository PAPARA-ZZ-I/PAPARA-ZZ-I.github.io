PAPARA(ZZ)I

Program for Annotation of Photographs and Rapid Analysis (of Zillions and Zillions) of Images

PAPARA(ZZ)I is an open-source program for annotating marine still imagery data. Images can be annotated using either free points, random points or gridded points. The compiled versions run under 64-bit Windows, Mac OS X and Linux systems. Alternativelly, PAPARA(ZZ)I can be run from MATLAB directly.
