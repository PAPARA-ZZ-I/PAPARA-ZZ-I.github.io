PAPARA(ZZ)I

Program for Annotation of Photographs And Rapid Analysis (of Zillions and Zillions) of Images

PAPARA(ZZ)I is an open-source program for annotating marine still imagery data. Images can be annotated using either free points, random points or gridded points. The compiled versions run under 64-bit Windows, Mac OS X and Linux systems. Alternatively, PAPARA(ZZ)I can be run from MATLAB directly.

Download

PAPARA(ZZ)I can be downloaded from:

  • GitHub: PAPARA(ZZ)I project repository (source code and web-installers of latest version)
  • Google Drive: PAPARA(ZZ)I MCR Installers (mcr-installers of latest version for installation on computers without internet access)
The latest version is 2.9. The installers for the latest version are always available for Windows first. The installers for Linux and MacOS may not always be available for the latest version.

Authors

PAPARA(ZZ)I was developed at the Alfred Wegener Institute, Helmholtz Centre for Polar and Marine Research (Germany) and the MARUM Center for Marine Environmental Sciences of the University of Bremen (Germany) by Yann Marcon and Autun Purser.

Support

The latest version of the PAPARA(ZZ)I user manual is provided with the program on the PAPARA(ZZ)I GitHub repository. Alternatively, a direct link to the user manual of PAPARA(ZZ)I v2.0 (old version) is available here.
