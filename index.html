PAPARA(ZZ)I

Program for Annotation of Photographs and Rapid Analysis (of Zillions and Zillions) of Images

PAPARA(ZZ)I is an open-source program for annotating marine still imagery data. Images can be annotated using either free points, random points or gridded points. The compiled versions run under 64-bit Windows, Mac OS X and Linux systems. Alternatively, PAPARA(ZZ)I can be run from MATLAB directly.

Download

PAPARA(ZZ)I can be downloaded from:

  • GitHub: PAPARA(ZZ)I project repository (source code and web-installers of latest version)
  • PANGAEA Data Publisher for Earth & Environmental Science: PAPARA(ZZ)I v1.6 (source code and web-installers of version 1.6)
  • Google Drive: PAPARA(ZZ)I MCR Installers (mcr-installers: recommended for installation on computers without internet access)

Authors

PAPARA(ZZ)I was developed at the Alfred Wegener Institute, Helmholtz Centre for Polar and Marine Research (Germany) by Yann Marcon and Autun Purser.

Support

The latest version of the PAPARA(ZZ)I user manual is provided with the program on the PAPARA(ZZ)I GitHub repository. Alternatively, a direct link to the user manual of PAPARA(ZZ)I v1.6 is available here.
